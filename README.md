**English**  
This mod ensures that fog in Minecraft remains **fair and vanilla-like**, even when using optimization mods like [Sodium Extra](https://modrinth.com/mod/sodium-extra).  
Many mods offer useful fog customization for visuals or performance, but some of those changes (like removing fog entirely) can **give players unfair advantages** in multiplayer.  
ExtraSodiumFogFix lets players **keep their favorite mods**, while **blocking only the fog tweaks that could break balance** ‚Äî like seeing through water, lava, or long distances.

‚úÖ No hard dependency on Sodium or Sodium Extra  
‚úÖ Works with any mod that touches fog ‚Äî by applying a mixin with **priority 1400**  
‚úÖ Keeps the game fair and honest without banning whole mods
